# 海之心通用预约系统


[查看演示](http://ty.ohyu.cn/)

### 为什么开源
到目前为止我们开发了很多系统，也有一部分系统开发出来后就闲置了，与其这样不如把低版本代码开源出来，这样既能满足小需求或者实力比较小的公司低成本使用，也能让更高需求的客户找到我们。

### 介绍
通用预约系统是成都海之心科技有限公司开发的一款适用性比较广的一款系统。目前开源的版本【预约项目+预约时间+表单+核销】适用觉大多数简单场景的预约。但是本套系统目前的实际的功能更多适用的范围更广。

之前这套系统做过很多行业的预约，学校，博物馆，图书馆等等。

### 安装部署

推荐服务器环境：PHP 7.3+ MYSQL5.6 + nginx

本系统采用PHP的Thinkphp 6框架开发。
* 1、导入目录里面的install.sql导入到mysql
* 2、修改目录里面的.env文件
* 3、打开后台 域名/admin/login/index 登录 

安装教程：https://www.ohyu.cn/index/help/detail/id/68

前端也开源了，前端使用uniapp开发，适配了微信公众号、微信小程序、手机H5 三端：https://github.com/ohyueo/generaluniapp
